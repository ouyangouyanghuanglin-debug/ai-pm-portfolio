## 🚀 快速开始

### 方式 1：直接在浏览器打开
1. 找到上面的原型链接
2. 右键 → 复制链接地址
3. 在浏览器中打开

### 方式 2：本地打开
```bash
# 进入作品集目录
cd /home/admin/.openclaw/workspace/作品集

# 用浏览器打开任意原型
# 项目 1
open 项目 1-AI 学习助手机器人/原型/index.html  # Mac
xdg-open 项目 1-AI 学习助手机器人/原型/index.html  # Linux
start 项目 1-AI 学习助手机器人/原型/index.html  # Windows

# 项目 2
open 项目 2-AI 周报生成工具/原型/index.html

# 项目 3
open 项目 3-小红书搜索 AI 重构/原型/index.html
```

### 方式 3：用 Python 启动本地服务器
```bash
cd /home/admin/.openclaw/workspace/作品集
python3 -m http.server 8000
# 然后访问 http://localhost:8000/项目 1-AI 学习助手机器人/原型/index.html
```

## 💡 面试使用指南

### 1. 作品集呈现
- **最佳方式**：将原型部署到线上（如 Vercel、Netlify）
- **备选方式**：面试时本地打开演示
- **简历写法**：附上作品集链接（GitHub/个人网站）

### 2. 介绍话术模板

**当被问到"你有 AI 产品经验吗？"：**
> "虽然没有正式实习，但我系统做了 3 个 AI 产品项目来学习。比如这个学习助手项目，我完整走了从调研到设计的全流程..."
> （然后打开原型演示）

**当被问到"你懂大模型应用吗？"：**
> "我做过一个基于 RAG 的学习助手项目。核心是用向量数据库存储用户资料，大模型基于检索结果生成答案..."
> （打开项目 1 原型展示对话功能）

**当被问到"你会怎么写 Prompt？"：**
> "我在周报助手项目里设计过 Prompt。核心是角色定义 + 任务描述 + 输出格式..."
> （打开项目 2 原型展示生成效果）

### 3. 每个项目重点展示

| 项目 | 展示重点 | 面试问题 |
|------|---------|---------|
| 项目 1 | RAG 技术理解、对话设计 | 大模型落地、技术架构 |
| 项目 2 | Prompt 工程、场景洞察 | AIGC 应用、商业化 |
| 项目 3 | 算法理解、大厂业务 | 推荐搜索、AI 重构能力 |
