# 项目 2：AI 周报生成工具「周报助手」

## 📌 项目定位

一款面向职场人的 AI 周报生成工具，基于大模型 + 工作数据整合，自动生成高质量周报。

---

## 🎯 产品背景

### 用户痛点
1. **写周报耗时**：每周花 1-2 小时整理工作、撰写周报
2. **内容难组织**：不知道如何结构化呈现工作成果
3. **数据难汇总**：需要从多个平台（Git、Jira、飞书等）收集数据
4. **价值难体现**：写了但领导看不到工作价值

### 市场机会
- 中国职场白领约 4 亿人
- 90% 以上公司要求写周报
- AI 办公产品处于爆发期

---

## 🎨 产品愿景

**让职场人从周报中解放，专注于真正有价值的工作**

---

## 📊 核心功能

### 功能 1：多源数据整合

| 数据源 | 整合内容 | 接入方式 |
|-------|---------|---------|
| Git | 提交记录、代码量 | API 授权 |
| Jira/飞书 | 任务完成情况 | API 授权 |
| 日历 | 会议记录 | API 授权 |
| 手动输入 | 临时工作、思考 | 文本输入 |

### 功能 2：AI 智能生成

**Prompt 设计示例：**
```
你是一名专业的职场助手，请根据以下工作内容生成周报：

【工作内容】
{用户输入的工作内容}

【要求】
1. 按"本周完成工作"、"遇到的问题"、"下周计划"结构组织
2. 突出工作价值和成果，用数据说话
3. 语言简洁专业，适合发送给领导
4. 字数控制在 500-800 字

【输出格式】
## 本周工作总结
...

## 遇到的问题与解决
...

## 下周工作计划
...
```

### 功能 3：模板中心

| 岗位类型 | 模板特点 | 适用人群 |
|---------|---------|---------|
| 技术岗 | 突出代码量、技术难点 | 开发、测试 |
| 产品岗 | 突出需求、数据变化 | 产品、运营 |
| 销售岗 | 突出业绩、客户进展 | 销售、BD |
| 管理岗 | 突出团队、战略进展 | 管理者 |
