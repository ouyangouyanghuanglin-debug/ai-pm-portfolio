# 项目1：AI 学习助手机器人「学伴」

## 📌 项目定位

一款面向大学生的 AI 学习助手，基于大模型 + RAG 技术，提供课程答疑、学习资料整理、复习计划生成等功能。

---

## 🎯 产品背景

### 用户痛点
1. **课后问题无人解答**：大学课堂容量大，老师难以及时回答每个学生问题
2. **学习资料分散**：PPT、笔记、作业分散在不同平台，整理耗时
3. **复习无规划**：不知道如何安排复习时间，考前突击效果差
4. **个性化需求强**：每个学生学习节奏、理解能力不同，需要个性化辅导

### 市场机会
- 全国在校大学生约 4000 万
- 在线教育市场规模超 3000 亿
- AI 教育产品处于早期爆发阶段

---

## 🎨 产品愿景

**让每个学生都拥有专属的 AI 学习伙伴**

## 🏗️ 技术架构

```
┌─────────────────────────────────────────────────────────┐
│                     用户层                              │
│   (微信小程序 / APP / Web)                              │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                     应用层                              │
│   对话管理 | 意图识别 | 上下文管理 | 会话存储           │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                     AI 层                               │
│   大模型 API (GLM-4/Qwen) | Prompt 工程 | 输出格式化    │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                     RAG 层                              │
│   向量数据库 | 知识检索 | 资料embedding | 相关性排序     │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                     数据层                              │
│   用户数据 | 学习记录 | 知识库 | 对话历史               │
└─────────────────────────────────────────────────────────┘
```
